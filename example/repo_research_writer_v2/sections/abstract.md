# Abstract

Scientific software development increasingly relies on complex computational pipelines, yet translating these repositories into publication-ready manuscripts remains labor-intensive, requiring 20-40 hours of manual effort to extract technical details, synthesize narrative structure, and integrate relevant literature. We present RRWrite, an AI-powered tool that automates manuscript generation from research code repositories through a multi-stage pipeline integrating repository analysis, literature search, evidence-based drafting, and adversarial critique. RRWrite analyzes repository structure, documentation, and Git history to extract technical evidence, searches PubMed and Semantic Scholar APIs to identify relevant citations, and generates journal-specific manuscripts with defense-in-depth validation ensuring all claims link to verifiable sources. The system employs section-specific drafting strategies with mandatory verification gates, automated citation formatting with DOI resolution, and a dual Git repository architecture separating tool development from manuscript versioning. We validated RRWrite through self-referential manuscript generation, where the system analyzed its own codebase and produced a complete 3,788-word manuscript with 23 DOI-verified citations across six sections in approximately 60 minutes. The workflow successfully identified repository architecture, integrated computational reproducibility literature [Wilkinson2016, Pimentel2023], and generated scientifically structured prose meeting PLOS Computational Biology formatting requirements. Automated critique detected 6 major and 8 minor issues, demonstrating quality control capabilities. RRWrite is available as an open-source Python package with Claude Code integration at https://github.com/USERNAME/rrwrite, reducing manuscript preparation time by 75% while maintaining evidence traceability and citation accuracy.
