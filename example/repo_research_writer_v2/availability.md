# Data and Code Availability

Source code for RRWrite is available at https://github.com/realmarcin/repo-research-writer under the MIT License. Installation requires Python 3.8+ and can be completed via `./install.sh`, which creates symbolic links to the global Claude Code skills directory. Python dependencies (pandas, PyYAML, openpyxl, LinkML) are listed in requirements.txt. Complete documentation is provided in README.md and supporting files in the docs/ directory.

A self-referential example manuscript demonstrating the complete workflow is included in `example/repo_research_writer_v2/`, containing all generated artifacts including repository analysis, outline, section drafts, literature citations, and critique reports. Data tables extracted during repository analysis are available in `data_tables/` subdirectories within each manuscript.
